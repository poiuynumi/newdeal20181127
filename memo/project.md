
기본 기능 : 회원가입, 회원 자유게시판, 채팅

맛집 SNS (맛집 아니어도 핫플레이스나 데이트코스 등으로 사용가능)
- 맛집 정보 공유 : 사진, 위치 정보, 가격 정보, 평가 등
- 좋아요 / 싫어요
- 인기 랭킹 보여주기
- 지역/메뉴 입력하면 등록된 정보 모아 보여주기
- 메뉴 추천 : 날씨나 기념일에 따른 메뉴추천 / 회원 정보나 활동에 근거한 추천


건강 관리 시스템
- 비만도 측정
- 음식에 따른 칼로리, 영양소 정보
- 운동에 따른 소비 칼로리 정보
- 위와 같이 캘린더 형식으로 섭취/소비 칼로리 보여주기 
- 식습관 패턴으로 결핍영양소와 해당 음식 추천해주기
- 신체 부위에 따른 효과적인 운동 추천


타로 사이트
- 랜덤으로 카드 뽑으면 카드의 의미 알려주기

- 관상 : 이목구비의 특징에 따른 관상 봐주기
